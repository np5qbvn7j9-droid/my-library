# مكتبتي 📚 — قاعدة المعرفة الشخصية

نظام إدارة معرفة شخصي (Second Brain): ملاحظات، أقسام ومجلدات بلا حدود، بحث سريع بالعربية، روابط ويكي `[[...]]` وخريطة معرفية، مراجعة بالتكرار المتباعد، اقتباسات، مراجع، قوالب، صندوق قراءة، خط زمني وإحصائيات.

- **محلي أولًا:** كل البيانات على جهازك (IndexedDB) — يعمل بدون إنترنت.
- **مزامنة اختيارية:** عبر Firebase بين جميع أجهزتك.
- **شخصي:** تسجيل الدخول مقفل على `gazzy155@gmail.com` فقط.

## 🚀 النشر على GitHub Pages (بدون أي أوامر طرفية)

جهازك لا يحتاج Node — GitHub يبني التطبيق تلقائيًا:

1. ادخل [github.com](https://github.com) بحساب gazzy155@gmail.com وأنشئ مستودعًا جديدًا (مثلًا `maktabati`) — اجعله **Private**.
2. ارفع **كل ملفات هذا المجلد** عبر "uploading an existing file" أو اسحب المجلد كاملًا (تأكد أن مجلد `.github/workflows` رُفع أيضًا — إن لم يظهر، أنشئ الملف يدويًا من "Add file → Create new file" باسم `.github/workflows/deploy.yml` وانسخ محتواه).
3. من إعدادات المستودع: **Settings → Pages → Source → GitHub Actions**.
4. انتظر دقيقتين حتى ينتهي البناء (تبويب **Actions**)، ثم افتح رابط التطبيق الظاهر هناك.
5. عند أي تعديل مستقبلًا: ارفع الملفات المعدلة فقط، وسيُعاد البناء والنشر تلقائيًا.

> ملاحظة: مستودع Private يتطلب أن تكون GitHub Pages متاحة (في الخطة المجانية Pages للمستودعات العامة فقط — إن أردت إبقاءه خاصًا مجانًا، اجعل المستودع Public؛ لا توجد بيانات شخصية في الكود، بياناتك تبقى في جهازك وFirebase).

## ☁️ تفعيل المزامنة (Firebase) — مرة واحدة

1. افتح [console.firebase.google.com](https://console.firebase.google.com) بحساب gazzy155@gmail.com → **Add project** (مثلًا `maktabati`) — عطّل Analytics.
2. من **Build → Firestore Database → Create database** → اختر Production mode.
3. من تبويب **Rules** في Firestore، الصق محتوى ملف `firestore.rules` الموجود في هذا المشروع ثم Publish.
4. من **Build → Authentication → Get started → Sign-in method** → فعّل **Google**.
5. من **Authentication → Settings → Authorized domains** → أضف نطاق تطبيقك (مثل `USERNAME.github.io`).
6. من ⚙️ **Project settings → Your apps → أيقونة `</>` (Web)** → سجّل التطبيق وانسخ كائن `firebaseConfig`.
7. افتح التطبيق → **الإعدادات** → الصق الإعدادات بصيغة JSON → **حفظ وتسجيل الدخول**.

بعدها تعمل المزامنة الفورية تلقائيًا بين الجوال والكمبيوتر.

## 📱 التثبيت على الجوال

افتح رابط التطبيق في Safari (آيفون) → مشاركة → **إضافة إلى الشاشة الرئيسية**. وفي أندرويد من Chrome → **تثبيت التطبيق**.

## 💾 النسخ الاحتياطي

من صفحة الإعدادات: **تصدير كل البيانات** ينزل ملف JSON كامل، و**استيراد نسخة** يدمجها مرة أخرى (الأحدث يفوز، لا حذف).

## 🧭 أساسيات الاستخدام

- **ربط الملاحظات:** اكتب `[[عنوان ملاحظة]]` داخل أي ملاحظة — تظهر تلقائيًا في «ملاحظات مرتبطة» وفي الخريطة المعرفية.
- **المراجعة المتباعدة:** من داخل الملاحظة اضغط 🔁 لإضافتها لنظام المراجعة، ثم راجع يوميًا من صفحة «المراجعة» وقيّم: نسيتها / بصعوبة / جيدة / ممتازة.
- **البحث الذكي:** يفهم عبارات مثل «ملاحظاتي هذا الشهر» ويتجاهل كلمات الحشو، ويطابق مع اختلاف التشكيل والهمزات.
- **الزر العائم +:** ملاحظة جديدة، من قالب، لصق من الحافظة، حفظ رابط، اقتباس.
